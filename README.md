# 🛡️ AI-Powered Network Intrusion Detection System

## 📌 Overview
This project is an AI-based Network Intrusion Detection System (NIDS) that detects cyber attacks using machine learning.

## 🚀 Features
- Real-time attack detection
- Risk scoring system
- Attack type classification (DDoS, Probe, etc.)
- Live monitoring dashboard
- Alert counter & logs

## 🧠 Technologies Used
- Python
- Scikit-learn (Random Forest)
- Streamlit (UI Dashboard)
- NumPy, Pandas

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run create.py

## 🎯 Output
- Detects Normal vs Attack
- Shows risk percentage
- Displays attack type
- Visualizes trends
